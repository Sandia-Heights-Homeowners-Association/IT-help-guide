# Quick Reference

Use this page for the essentials only.

## If You Remember Only 5 Things

1. Accept your Microsoft invitation email first.
2. Use SharePoint links, not file attachments, for committee docs.
3. Do not share files directly to committee groups for external-member committees.
4. If you have a role inbox, reply as that role (or CC it) for SHHA business.
5. Contact **ITHelp@sandiahomeowners.org** when stuck.

## Core Access Points

- **SharePoint links**: primary location for committee files
- **Outlook**: used by staff and role mailbox users
- **Google Calendar reminders**: used for Board scheduling notifications

## Common Actions

### Share a file with your committee

1. Upload file to SharePoint
2. Copy SharePoint link
3. Email link to committee mailing list

### Confirm your access setup

- Confirm invitation was accepted
- Confirm you can open the committee SharePoint link
- Confirm you receive committee group emails

## Contact

- IT support: **ITHelp@sandiahomeowners.org**
- For membership updates, committee chairs should coordinate with office staff
